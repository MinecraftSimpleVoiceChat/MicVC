    Step 1: Install Minecraft Forge (Automatic setup may handle this for Windows users)

    Begin by installing Minecraft Forge, which is essential for running mods. Select the Forge version that matches your Minecraft version for best compatibility. If you've used the automatic setup in Step 2 (Windows users), it might have already installed Forge. Please verify if Forge is installed, and if not, proceed with the manual installation from the provided link.

    Step 3: Locate Minecraft Folder (Automatic setup simplifies this for Windows users)

    Locating your Minecraft installation folder is a key step for manual mod installation. Here’s how to find it on different operating systems. For Windows users, the automatic setup in Step 2 may simplify accessing this directory, but understanding how to find it manually is still beneficial for all users and for other operating systems.

    • Windows: Open File Explorer, type %appdata%\.minecraft in the address bar, and press Enter.
    • macOS: Open Finder, press Cmd + Shift + G, and enter ~/Library/Application Support/minecraft.
    • Linux: Open your Home folder, and navigate to .minecraft. If it's hidden, press Ctrl + H.

    Step 4: Access or Create the Mods Folder (Automatic setup aims to ensure 'mods' folder exists on Windows)

    Once you've located your Minecraft folder, the next step is to find or create the 'mods' folder. If you don’t see a folder named 'mods', you need to create one. This is where you will place the Simple Voice Chat mod and any other mods you wish to install. For Windows users, the automatic setup in Step 2 is designed to ensure these essential folders are correctly set up, including the 'mods' folder. However, verifying its presence is always a good practice.

    Simple Voice Chat Reminder: Regardless of whether you use automatic setup or manual steps, always ensure that no other voice chat mods are present in the 'mods' folder to prevent potential conflicts.

    Step 5: Install Simple Voice Chat and Other Mods (Essential for all users, automatic setup does not handle this)

It’s crucial to ensure that all mods are compatible with your specific Minecraft version and the installed version of Forge. Note that the automatic setup in Step 2 does not handle the installation of individual mods; this step is required for all users, regardless of whether automatic setup was used.

    Step 6: Launch Minecraft with Forge Profile (Final step for all installation types)

    The final step is to launch Minecraft using the Forge profile. This action will start Minecraft with Minecraft Forge and all the mods you have installed, including Simple Voice Chat. Whether you opted for the automatic setup in Step 2 or followed the manual steps, launching with the Forge profile is the concluding action for all users.

    1. Open the Minecraft Launcher to get started.
    2. In the Minecraft Launcher, find the profile selection dropdown menu (usually located before the 'Play' button) and select the Forge profile.
    3. If you do not see the Forge profile listed, it may be necessary to reinstall Minecraft Forge, paying close attention to the profile installation steps during the setup process.
    4. Once the Forge profile is selected, click 'Play' to launch Minecraft. This will load Minecraft with Forge, along with Simple Voice Chat and any other mods you have added to your 'mods' folder.

    Simple Voice Chat Configuration: After Minecraft has launched, you can configure Simple Voice Chat settings within the game. Go to Options, then Music & Sounds, to adjust settings like voice activation mode, volume levels, and other audio preferences to best suit your needs.
